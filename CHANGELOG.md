# Changelog

All notable changes to `3d-ref-skills` are documented here.

## [2.0.0] — 2026-06-20 — Initial Public Release

### Added
- `ref-brief` — Complete pre-production reference document generator for any 3D asset
- `ref-style-decode` — 8-dimension style deconstruction framework
- `ref-audit` — Mid-modeling reference board audit with asset-type-specific checks
- `ref-marketplace` — Dual modeling + marketing reference brief for marketplace publishing
- `ref-client` — Client-facing scope and visual alignment document for freelance work
- `docs/sources.md` — Curated reference source directory for 3D artists
- Full example outputs for all 5 skills

### Works with
- Claude Code, Cursor, Codex CLI, Gemini CLI
- Maya, Blender, ZBrush, 3ds Max, Houdini, Cinema 4D
- UE5, Unity, Godot, web, film, archviz
