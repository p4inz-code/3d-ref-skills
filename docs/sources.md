# Curated Reference Sources for 3D Artists

> Part of `3d-ref-skills` — github.com/p4inz-code/3d-ref-skills
> Updated: 2026-06-20 | Version: 2.0.0

A curated, opinionated list of where to find the best references for 3D work.
Organized by what you're looking for, not by platform.

---

## For Silhouette and Shape Reference

| Source | What It's Best For | Notes |
|--------|-------------------|-------|
| [ArtStation](https://artstation.com) | Style-matched 3D and concept art | Filter by "3D" to find real mesh references |
| [Pinterest](https://pinterest.com) | Broad visual gather, mood boards | Quantity over quality — cross-reference anything critical |
| [Behance](https://behance.net) | Product design, industrial design shape language | Excellent for hard surface silhouettes |
| [Google Images](https://images.google.com) | Real-world photography | Filter by "Large" for detail shots |
| [Sketchfab](https://sketchfab.com) | Existing 3D assets — inspect topology decisions | Download free assets to study mesh and UV approach |
| [Concept Art World](https://conceptartworld.com) | Film and game concept art | Official concept art releases from AAA productions |

## For Orthographic Blueprints

| Source | What It's Best For |
|--------|-------------------|
| [The-Blueprints.com](https://the-blueprints.com) | Vehicles — cars, aircraft, ships, military hardware |
| [WW2 Drawings](https://ww2drawings.jexiste.fr) | WWII era military vehicle blueprints (free) |
| US Army Field Manuals (public domain) | Military equipment, weapons, crates, gear |
| [WikiMedia Commons](https://commons.wikimedia.org) | Historical technical drawings (public domain) |
| Patent drawings (Google Patents) | Industrial objects, tools, mechanisms — exact dimensions |
| Manufacturer spec sheets | Modern products — often available on brand websites |
| [Archive.org](https://archive.org) | Historical books with technical illustrations |

## For Material and Surface Macro Photography

| Source | What It's Best For | License |
|--------|-------------------|---------|
| [Texture.com](https://texture.com) | PBR textures + macro surface photography | Free for reference |
| [Polyhaven](https://polyhaven.com) | CC0 textures and HDRIs | Fully free, commercial use |
| [AmbientCG](https://ambientcg.com) | CC0 PBR material library | Fully free |
| [Unsplash](https://unsplash.com) | High-quality real-world photography | Free commercial use |
| [Pexels](https://pexels.com) | Same as Unsplash, different library | Free commercial use |
| [Flickr Creative Commons](https://flickr.com/creativecommons) | Specialized photography, macro detail | Check individual license |
| Your own phone camera | Best macro photography you can get | You own it — use it |

## Reference Tools

| Tool | What It Does | Price |
|------|-------------|-------|
| [PureRef](https://pureref.com) | Desktop reference board, drag-and-drop, overlay on DCC | Pay what you want |
| [Milanote](https://milanote.com) | Web-based, shareable boards — good for client alignment | Free tier / paid |
| [Miro](https://miro.com) | Collaborative boards for team reference sharing | Free tier / paid |
| [Adobe Bridge](https://adobe.com/bridge) | Local file-based reference management | Included in CC |
| Kuadro | Simple, lightweight reference viewer (Windows) | Free |
| [Eagle](https://eagle.cool) | Local image library manager with tags and search | Paid, worth it for large reference libraries |
