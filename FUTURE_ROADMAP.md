## v3.0.0 — Character and Specialist Release

New Skills:
- [ ] `ref-character` — Character reference deep-dive
  - Facial topology reference framework
  - Anatomy and structure reference (not just surface)
  - Deformation zone reference for rigged characters
  - Costume and material layering reference
  - SSS zone reference for skin

- [ ] `ref-archviz` — Architecture and visualization reference
  - Building material reference (concrete, brick, glass, steel, cladding)
  - Interior lighting reference
  - Human scale reference for architecture
  - Client presentation reference

- [ ] `ref-hardsurface` — Hard surface specialist
  - Panel line system reference
  - Greebling density and scale reference
  - Industrial design language reference
  - Bevel width calibration reference

New Documentation:
- [ ] `docs/VFX_REFERENCE_WORKFLOW.md` — Standalone VFX reference guide
- [ ] `docs/TEXTURE_REFERENCE_WORKFLOW.md` — Standalone texture reference guide
- [ ] Move planning docs to `docs/meta/` to clean up root level

## Community-Driven (no scheduled version)

These skills will be built when contributors from those communities submit them:

- `ref-substance` — Substance Designer-specific graph reference
- `ref-zbrush` — ZBrush sculptural reference (plane breaks, form language)
- `ref-anime` — Anime/stylized toon reference workflow
- `ref-lowpoly` — Low-poly and mobile reference (N64/PSX aesthetic, mobile budgets)

## Contribution Opportunities

If you want to contribute a skill, see CONTRIBUTING.md.
The highest-impact unbuilt skills in priority order:

1. `ref-character` — Large community, high demand, ref-audit already has C1–C3 checks to build from
2. `ref-archviz` — Archviz community is active and underserved by AI tooling
3. `ref-hardsurface` — Hard surface is the largest single category on Fab
